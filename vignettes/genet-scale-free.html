Generating scale-free networks

In this vignette, we show how to generate graphs with an overlapping community structure and a target degree sequence drawn from a power-law distribution.

library(genet)

First, we set set the number of communities K and types L, and fix a target community structure. Depending on the application of interest, users can input their custom community structure. Here, for simplicity, we will generate a balanced structure via the generate_balanced_interactions function. This function randomly assigns each type to K_each communities and then compute the resulting type interaction matrix. The community memberships are assigned such that: no types have identical assignments; all communities have at least one type; and the resulting graph of type interactions is connected.

K <- 100
L <- 200
ints <- generate_balanced_interactions(K, L, K_each = 5)$interaction_matrix

Next, we draw the target degree centrality sequence from a power-law degree sequence with the following parameters:

d_min <- 3
d_max <- 500
alpha <- 1.937

We set the minimum and maximum degree to 3 and 500, and the power law’s exponent to 1.937 so that the expected degree is approximately 15.

as.numeric(dpower_law(3:500, alpha, d_min, d_max)  %*% 3:500)
#> [1] 15.00011

To more closely reproduce the power-law distribution, we set the target degrees equal to the \((L + 1)\)-quantiles of the distribution and then adjust the type cardinalities so that the empirical distribution complies with the power law.

target_degrees <- qpower_law(seq(0, 1, length.out = L), alpha, d_min, d_max)
probs <- dpower_law(target_degrees, alpha, d_min, d_max)
ratios <- probs / min(probs)
cards <- round(ratios * 2)

We can now generate a random network.

g <- genet(1, ints, cards, dc = T, target_centrality = target_degrees)[[1]]

The output of genet is a list of igraph graph objects.

igraph::vcount(g)
#> [1] 148611
igraph::ecount(g)
#> [1] 906618

We can use the igraph::degree function to extract the empirical degree distribution of the graph.

degree_distr <- table(igraph::degree(g))
